# Building Operations AI Assistant

An intelligent, multi-agent AI assistant designed to optimize buildiung operations

## Architecture 
* **Orchestration:** LangGraph
* **Retrieval Engine:** Azure AI Search (Hybrid Search)
* **LLM & Embeddings:** Azure OpenAI (gpt-4o & text-embedding-3-small)
* **Backend:** FastAPI
